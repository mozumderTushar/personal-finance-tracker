# Budget Dashboard

A Google Sheets-based personal finance dashboard to track monthly income, expenses, and balance with automated summaries and trend charts.

---

## Features

- **Automatic Month Extraction**  
  Converts transaction dates into month-year format (e.g., Nov-2025) automatically.

- **Monthly Summary Table**  
  Calculates total Income, Expenses, and Balance for each month using `SUMIFS`.

- **Trend Charts**  
  Visualizes Income, Expenses, and Balance trends over time with line or combo charts.

- **Conditional Formatting**  
  Highlights over-budget categories or unusual expenses automatically.

- **Clean and Automated**  
  Easy-to-update formulas and dynamic tables make financial tracking simple.

---

## How to Use

1. **Add Transactions**  
   Fill in your transaction data in the table with columns:
   - Date  
   - Type (Income / Expense)  
   - Categories  
   - Amount  
   - Descriptions

2. **Monthly Summary**  
   The dashboard automatically pulls monthly totals for Income, Expenses, and Balance.

3. **Trend Charts**  
   Charts update automatically when new transactions are added.

4. **Customize**  
   Adjust categories, add savings goals, or add recurring bills sections to suit your needs.
