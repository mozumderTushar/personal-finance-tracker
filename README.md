This project is a Google Sheets-based personal finance dashboard designed to help you track and visualize your income, expenses, and savings over time. Features include:

Automatic extraction of month and year from transaction dates

Monthly summaries of Income, Expenses, and Balance

Trend charts for Income, Expenses, and Balance

Conditional formatting for easy tracking

Clean, automated, and easy-to-update Google Sheets formulas

Suitable for personal budgeting and financial planning
